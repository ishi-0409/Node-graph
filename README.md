# Node Network Graph Tool

テキストを「ノード」として整理し、その関係性を「ネットワーク図」として可視化・保存する個人向けのWebアプリケーションです。  
Gemini API を用いて、蓄積されたノード同士の「隠れた関係性」をAIが自動で推測・提案する先進的な機能を搭載しています。

---

## 💡 コンセプト
文章のアイデア整理、登場人物の相関図作成、シナリオのプロット構築など、頭の中の雑多なテキスト情報（ノード）を視覚的にマッピングし、AIとの共同作業によって新しい繋がり（エッジ）を発見・整理することができます。

---

## ✨ 主な機能

### 1. 多階層フォルダ管理 (キャンバスの切り替え)
* サイドバーでフォルダを新規作成し、ツリー構造で管理できます。
* 各フォルダが独立したネットワーク図（キャンバス）を保持するため、テーマごとにワークスペースを切り替えられます。

### 2. 直感的なキャンバス操作 (React Flow)
* **ノードの追加:** キャンバスの何もない場所を **ダブルクリック** するだけで、素早くテキストノードを作成できます。
* **自由な配置:** 作成したノードはドラッグ＆ドロップで自由に動かせます（配置位置は自動で即座に保存されます）。
* **編集・削除:** ノードにカーソルを合わせると表示されるアイコンから、テキストの編集やノードの削除が可能です。

### 3. 関係性の接続とラベリング (エッジ)
* ノードの右端（ソース）から別のノードの左端（ターゲット）へドラッグすることで、関係性の線を繋げられます。
* 接続時に「対立する概念」「原因と結果」「補足説明」などの**カスタムラベル**を自由に設定できます。
* 繋いだ線（エッジ）をクリックして、ラベルの編集や関係性の削除を行うことができます。

### 4. Gemini AI による関係性の提案
* 右上の「関係性を探す (AI)」ボタンを押すと、キャンバス内のまだ繋がっていないノード群を Gemini が分析します。
* 意味的・文脈的に関連性の高いノードのペアと、その理由（ラベル）をAIが自動で提案します。
* 提案カードに対して「承認」を押すと自動でキャンバス上に接続線が追加され、「却下」すると提案をスキップします。

### 5. ソーシャルログイン認証 (Supabase Auth)
* Google または GitHub アカウントを使用したセキュアなサインインに対応。
* ユーザーごとにデータが完全に分離され、安全に自分だけのグラフデータを保存できます。

## 🛠 テクノロジー・スタック

### フロントエンド
* **フレームワーク:** React + TypeScript (Vite)
* **グラフ描画:** React Flow (`@xyflow/react`)
* **スタイリング:** Vanilla CSS (モダンなダークテーマ & ガラスモフィズムデザイン)
* **ホスティング:** Vercel

### バックエンド
* **APIフレームワーク:** FastAPI (Python)
* **AI連携:** Google GenAI SDK (`gemini-3.5-flash` モデル)
* **認証・トークン検証:** PyJWT
* **ホスティング:** Google Cloud Run (サーバーレス)

### データベース / インフラ
* **データベース:** Supabase (PostgreSQL)
* **認証管理:** Supabase Auth
